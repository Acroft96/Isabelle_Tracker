Create new branch
git checkout -b updatedIsabelleFiles 

stage > commit > push
git add .
git commit -m "Updated files for Isabelle Tracker project"
git push -u origin updatedIsabelleFiles

Back to branch for pulling and pushing 
git branch (verfiys which branch you are on)
git checkout main (or whatever branch you are wanting to pull from and push to)

Undo your last commit
git reset --soft HEAD~1
